# Digital Image Processing Fundamentals
This repository contains my homework assignments for the Digital Image Processing course.

## Contents
1. **Image Resampling**: Implementation of Downsampling and Upsampling techniques from scratch using NumPy and Matplotlib.
2. **Data Visualization**: Advanced image data visualization using Matplotlib subplots and grayscale analysis.

## Tools Used
* Python 3
* NumPy
* Matplotlib
* Google Colab
